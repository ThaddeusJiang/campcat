## Usage

1. Send a message to https://t.me/campcatbot
2. Just wait for the available days

![campcatbot QRCode](./docs/campcatbot.JPG)

## Develop

install

```sh
npm install
```

crawl

```
npm start
```

send message to telegram

```
export TELEGRAM_TOKEN=<token>

npm run send:telegram
```
